## Zellij Alt key fix for macOS

**Thread:** Zellij keybindings all use Alt as the modifier, but macOS Option keys produce Unicode characters instead of Alt escape sequences — so none of the bindings work out of the box.

Researched the recommended approach. Three options: configure the terminal to remap Option→Alt, change Zellij's secondary modifier, or rebind all keys to Ctrl. Terminal-level fix was clearly best — one line vs. rebinding 30+ keys with collision risk. Set `option_as_alt = "OnlyLeft"` in `mac/alacritty.toml` so Left Option sends Alt for Zellij while Right Option stays available for special characters.

**Open:** Change is uncommitted. Needs Alacritty restart to take effect.
